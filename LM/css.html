Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML.

Con CSS, podemos controlar el diseño, la disposición y la apariencia de los elementos en una página web.

Sintaxis de CSS

La sintaxis de CSS está compuesta por selectores y declaraciones.

Un selector apunta al elemento HTML que quieres estilizar. Una declaración es una propiedad y su valor.


/* Ejemplo de sintaxis CSS */
selector {
    propiedad: valor;
}

Selectores en CSS

CSS ofrece una variedad de selectores para apuntar a los elementos HTML que queremos estilizar:

  • Selector de etiqueta: Selecciona todos los elementos de un tipo específico. Ej: p { color: red; }
  • Selector de clase: Selecciona elementos con una clase específica. Ej: .clase { color: blue; }
  • Selector de ID: Selecciona un elemento con un ID específico. Ej: #id { color: green; }
  • Selector de atributo: Selecciona elementos con un atributo específico. Ej: [type="text"] { color: purple; }
  • Selector de pseudo-clase: Selecciona elementos en un estado específico. Ej: a:hover { color: orange; }
  • Selector de pseudo-elemento: Selecciona una parte específica de un elemento. Ej: p::first-line { color: pink; }

Propiedades CSS

Propiedades Relativas a las Fuentes

  • font-family: Define la familia de la fuente. font-family: 'Arial', sans-serif;
  • font-size: Define el tamaño de la fuente. font-size: 16px;
  • font-weight: Define el grosor de la fuente. font-weight: bold;
  • font-style: Define el estilo de la fuente (normal, cursiva, etc.). font-style: italic;
  • line-height: Define la altura de la línea. Ej: line-height: 1.5;
  • text-align: Alinea el texto. text-align: center;
  • text-decoration: Define la decoración del texto (subrayado, tachado, etc.). text-decoration: underline;
  • text-transform: Controla la capitalización del texto. text-transform: uppercase;

Propiedades Relativas al Fondo

  • background-color: Define el color de fondo. background-color: lightblue;
  • background-image: Define una imagen de fondo. background-image: url('imagen.jpg');
  • background-repeat: Define cómo se repite la imagen de fondo. background-repeat: no-repeat;
  • background-size: Define el tamaño de la imagen de fondo. Ej: background-size: cover;
  • background-position: Define la posición de la imagen de fondo. Ej: background-position: center;

Cómo Enlazar HTML y CSS

Para aplicar estilos CSS a un documento HTML, necesitamos enlazar ambos archivos. Hay varias formas de hacerlo:

1. Usando la Etiqueta <link>en el <head>

La forma más común y recomendada es incluir una hoja de estilos externa usando la etiqueta <link> en la sección lt;head>de tu documento HTML. Esto permite mantener el HTML y CSS separados, facilitando el mantenimiento y la gestión de los estilos.


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página Web</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Bienvenido a mi página web</h1>
    <p>Este es un ejemplo de cómo enlazar HTML y CSS.</p>
</body>
</html>

2. Usando la Etiqueta <style> en el <head>

Otra forma de añadir estilos CSS es mediante la etiqueta <style> directamente en el <head> del documento HTML. Esta técnica es útil para aplicar estilos rápidos o específicos que no justifiquen la creación de un archivo CSS separado.


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página Web</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
        }
        h1 {
            color: #333;
        }
    </style>
</head>
<body>
    <h1>Bienvenido a mi página web</h1>
    <p>Este es un ejemplo de cómo usar la etiqueta <style> para CSS en el HTML.</p>
</body>
</html>

3. Usando Atributos de Estilo en las Etiquetas HTML

También puedes aplicar estilos CSS directamente a los elementos HTML usando el atributo <style> Esta práctica se conoce como estilos en línea y no se recomienda para grandes proyectos debido a la dificultad para mantener y modificar los estilos.


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página Web</title>
</head>
<body>
    <h1 style="color: blue;">Bienvenido a mi página web</h1>
    <p style="font-size: 16px;">Ejemplo de  estilos en línea en HTML.</p>
</body>
</html>

Modelo de caja (Box Model)

El modelo de caja es un concepto fundamental en CSS. Describe cómo se estructuran los elementos en la página web, permitiendo manipular márgenes, bordes, relleno y contenido.

El modelo de caja consta de cuatro partes:

  • Contenido: el área donde se muestra el contenido (texto, imágenes, etc.).
  • Relleno (padding): el espacio entre el contenido y el borde.
  • Borde (border): el contorno de la caja.
  • Margen (margin): el espacio entre la caja y otros elementos.
.elemento {
    width: 200px;
    padding: 10px;
    border: 5px solid #333;
    margin: 20px;
}

En el ejemplo anterior, tenemos una caja con un ancho de 200px, un relleno de 10px, un borde de 5px y un margen de 20px. Estos valores determinan el tamaño y el espaciado de la caja.

Bordes y sus propiedades

Las propiedades de borde en CSS permiten agregar un contorno a los elementos. Algunas de las propiedades principales incluyen:

  • border-width: define el grosor del borde.
  • border-style: define el estilo del borde (solid, dotted, dashed, etc.).
  • border-color: define el color del borde.
.elemento-borde {
    border-width: 4px;
    border-style: solid;
    border-color: #007bff;
}

También se puede utilizar border-radius para redondear los bordes, como en el siguiente ejemplo:

.elemento-borde-redondeado {
    border: 2px solid #007bff;
    border-radius: 10px;
}

Pseudo-clases en CSS

Las pseudoclases en CSS son palabras clave que se agregan a los selectores para aplicar estilos en un estado específico de un elemento, como cuando el usuario pasa el cursor sobre un elemento o un enlace ha sido visitado.

:hover

La pseudoclase :hover se activa cuando el usuario coloca el cursor sobre un elemento. Es útil para enlaces y botones.

    text-decoration: none;
    border-radius: 5px;
    transition: background-color 0.3s;
}

.btn-hover:hover {
    background-color: #2980b9;
}

:active

La pseudoclase :active se aplica a un elemento mientras es presionado, ideal para botones.

Definiciones en Función del Contexto

Las definiciones en función del contexto permiten aplicar estilos basados en la posición de los elementos dentro de la estructura HTML.

  • Selector descendiente: Selecciona los elementos dentro de un contenedor específico. Ej: div p { color: blue; }
  • Selector hijo: Selecciona los elementos que son hijos directos de un contenedor específico. Ej: div > p { color: green; }
  • Selector hermano adyacente: Selecciona un elemento que es el siguiente hermano inmediato de otro. Ej: h1 + p { margin-top: 0; }
  • Selector hermano general: Selecciona todos los elementos hermanos de un elemento anterior. Ej: h1 ~ p { color: red; }

Selector de Hijo Directo >

Este selector selecciona solo los hijos directos de un elemento. Esto permite aplicar estilos solo a los elementos de un nivel inmediato dentro de otro.

/* Aplica estilo solo a los párrafos que son hijos directos de un div */
div > p {
    margin-bottom: 10px;
}

En este caso, solo los <p> que están directamente dentro de un <div> serán estilizados.

Selector de Hermano Adyacente +

Este selector aplica estilos al primer elemento que sigue inmediatamente a otro elemento especificado.

/* Aplica estilo al párrafo que sigue a un h2 */
h2 + p {
    color: #555;
    font-style: italic;
}

En este caso, solo el primer <p> inmediatamente después de un <h2> se verá afectado.

Selector de Hermano General ~

Este selector afecta a todos los elementos hermanos que comparten el mismo padre y vienen después de un elemento especificado.

/* Estilo aplicado a todos los párrafos que siguen a un h2 */
h2 ~ p {
    color: #007bff;
}

A diferencia de +, este selector aplica estilos a todos los elementos hermanos en lugar de solo al inmediato.

Metodología BEM (Block, Element, Modifier)

BEM (Block, Element, Modifier) es una metodología de nomenclatura para CSS que facilita la organización y escalabilidad del código al dividir la interfaz en componentes reutilizables y predecibles.

Conceptos Clave de BEM

  • Block (Bloque): Es el contenedor principal que representa un componente de la interfaz. Es independiente y puede reutilizarse. Un "block" puede contener elementos y modificadores.
    Ejemplo de un block: <div class="menu">...</div>
  • Element (Elemento): Es una parte de un bloque que no puede existir sin el bloque padre. Se utiliza para describir componentes internos de un bloque. Los elementos están asociados a su bloque utilizando un doble guion bajo (`__`).
    Ejemplo de un block y sus elementos: <div class="menu"> <div class="menu__item">...</div> <div class="menu__link">...</div> </div>
  • Modifier (Modificador): Es una variación de un bloque o un elemento. Los modificadores se usan para representar diferentes estados, estilos o variaciones del bloque o elemento. Se indican con un doble guion (`--`).
    Ejemplo de un modificador: <div class="button button--primary">Enviar</div>

Cómo nombrar las clases en BEM

La nomenclatura en BEM sigue una estructura muy clara:

  • Bloque: El nombre de un bloque es el componente principal de la interfaz.
    Ejemplo: .header .menu .form
  • Elemento: El nombre de un elemento siempre está vinculado a un bloque, separado por un doble guion bajo (`__`).
    Ejemplo: .menu__item .form__input
  • Modificador: Los modificadores representan variaciones y se añaden a los bloques o elementos con un doble guion (`--`).
    Ejemplo: .button--large .header--dark

Ejemplo Completo en HTML y CSS con BEM

Vamos a aplicar BEM para crear una tarjeta (card) con un encabezado, cuerpo, y botón:

    <div class="card">
        <header class="card__header">
            <h1 class="card__title">Título de la tarjeta</h1>
        </header>
        <div class="card__body">
            <p class="card__text">Este es el contenido de la tarjeta.</p>
        </div>
        <footer class="card__footer">
            <button class="card__button card__button--primary">Leer más</button>
        </footer>
    </div>

    .card {
        border: 1px solid #ccc;
        border-radius: 5px;
        padding: 20px;
    }

    .card__header {
        font-size: 1.5em;
        font-weight: bold;
    }

    .card__body {
        margin-top: 15px;
    }

    .card__footer {
        margin-top: 20px;
    }

    .card__button {
        padding: 10px 15px;
        border: none;
        cursor: pointer;
    }

    .card__button--primary {
        background-color: blue;
        color: white;
    }

Beneficios de BEM

La metodología BEM ofrece las siguientes ventajas:

  • Escalabilidad: Facilita el crecimiento del proyecto sin que el CSS se vuelva difícil de gestionar.
  • Reusabilidad: Los bloques y elementos son fácilmente reutilizables en distintas partes de la aplicación.
  • Modularidad: Cada bloque es independiente, lo que permite realizar cambios sin afectar a otros componentes.
  • Mantenibilidad: La estructura clara y consistente ayuda a los desarrolladores a comprender y modificar el código rápidamente.

Otras metodologías de nomenclatura para CSS

Además de BEM, existen otras metodologías que ayudan a organizar el CSS de forma eficiente:

  • SMACSS (Scalable and Modular Architecture for CSS): Se centra en la modularización dividiendo el CSS en cinco categorías: Base, Layout, Module, State, Theme.
  • OOCSS (Object-Oriented CSS): Promueve la separación de estructura y apariencia, facilitando la reutilización de objetos CSS.
  • Atomic CSS: Utiliza clases pequeñas y especializadas para describir propiedades específicas, promoviendo la reutilización aunque a costa de muchos nombres de clase.

Posicionamiento en CSS

El posicionamiento en CSS es fundamental para controlar el layout de una página web. Permite definir cómo y dónde deben colocarse los elementos dentro del espacio disponible en la página. A lo largo del tiempo, las técnicas de posicionamiento han evolucionado, desde el uso de position (absoluto, relativo, fijo y estático) hasta técnicas modernas como Flexbox y CSS Grid, que simplifican mucho el trabajo con layouts responsivos.

Tipos de Posicionamiento en CSS

  • static: Es el valor por defecto. Los elementos se posicionan en el flujo natural del documento.
  • relative: Permite mover el elemento desde su posición original en el flujo del documento sin modificar el espacio que ocupa.
  • absolute: Posiciona un elemento respecto a su contenedor más cercano con position: relative o absolute.
  • fixed: Fija el elemento en una posición específica de la pantalla, independientemente del scroll.
  • sticky: Es una combinación de relative y fixed. El elemento se comporta como relative hasta que se activa en una posición, y luego permanece fijo.

Ejemplos básicos de posicionamiento:

position: relative mueve el elemento respecto a su posición natural:


.elemento-relativo {
  position: relative;
  top: 10px; /* El elemento se moverá 10px hacia abajo */
  left: 20px; /* El elemento se moverá 20px hacia la derecha */
}

position: absolute saca al elemento del flujo normal del documento:


.elemento-absoluto {
  position: absolute;
  top: 50px;  /* Se coloca a 50px del borde superior del contenedor */
  left: 30px; /* Se coloca a 30px del borde izquierdo del contenedor */
}

Para que un elemento con position: absolute funcione, debe estar dentro de un contenedor con position: relative o absolute.

Flexbox

El modelo de caja flexible, o Flexbox, es ideal para diseños unidimensionales (ya sea en fila o columna). Es muy útil para distribuir espacio entre elementos y alinear ítems dentro de un contenedor.

Conceptos clave en Flexbox

  • justify-content: Controla la alineación de los ítems a lo largo del eje principal (horizontal o vertical).
  • align-items: Alinea los ítems en el eje perpendicular (transversal).
  • flex-direction: Define si los ítems se alinean en fila (row) o en columna (column).
  • flex-wrap: Controla si los ítems deben mantenerse en una sola fila o permitir que envuelvan en múltiples filas.

Ejemplo básico de Flexbox:


.container {
  display: flex;
  justify-content: space-between; /* Espacio igual entre ítems */
  align-items: center; /* Centrado vertical de los ítems */
  flex-wrap: wrap; /* Permite que los ítems envuelvan a la siguiente línea si es necesario */
}

Este código crea una distribución flexible donde los ítems dentro del contenedor .container se alinearán con espacio entre ellos y se centrarán verticalmente.

Distribución flexible con flex-grow y flex-shrink

El uso de flex-grow y flex-shrink permite controlar cuánto debe crecer o encoger un elemento dentro de un contenedor flex. Por ejemplo:


.item {
  flex-grow: 1; /* Permite que este ítem crezca para ocupar más espacio disponible */
  flex-shrink: 2; /* Este ítem encogerá dos veces más rápido que otros cuando el contenedor sea más pequeño */
}

¿Qué es CSS Grid?

CSS Grid es un sistema de diseño bidimensional que divide una página en celdas o áreas, permitiendo una gran precisión en la disposición de los elementos. A diferencia de Flexbox, que es un diseño unidimensional (columnas o filas), CSS Grid permite trabajar con ambas dimensiones a la vez.

Propiedades clave de CSS Grid

1. Contenedor Grid

Para activar el modo Grid en un contenedor, utilizamos la propiedad display: grid;:

.grid-container {
    display: grid;
}

2. Definición de columnas y filas

La disposición de columnas y filas se controla principalmente con las propiedades grid-template-columns y grid-template-rows. Aquí tienes un ejemplo sencillo:

.grid-container {
    display: grid;
    grid-template-columns: 200px 200px;
    grid-template-rows: 100px 100px;
}

Este ejemplo crea una cuadrícula de dos columnas de 200px y dos filas de 100px.

3. Espaciado entre celdas

Para añadir espacio entre las celdas del grid, usa grid-gap:

.grid-container {
    display: grid;
    grid-template-columns: 200px 200px;
    grid-template-rows: 100px 100px;
    grid-gap: 10px;
}

4. Áreas de Grid

Las áreas de grid permiten definir bloques de celdas en los que se puede colocar contenido. Se utilizan en combinación con la propiedad grid-template-areas:

.grid-container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

5. Colocación de Elementos

La colocación se realiza a través de propiedades como grid-column y grid-row, que permiten controlar en qué columna y fila se sitúa cada elemento:

.item {
    grid-column: 1 / 3;
    grid-row: 2 / 4;
}

6. Unidades de Medida

Las unidades como fr (fracción), px, %, y auto permiten gran flexibilidad para la distribución de los elementos en el grid.

.grid-container {
    grid-template-columns: 1fr 2fr 1fr;
}

En este caso, las columnas estarán en una proporción de 1:2:1.

Ejemplo Completo

Aquí tienes un ejemplo completo de una estructura de CSS Grid con un contenedor y algunos elementos:

<div class="grid-container">
    <div class="header">Header</div>
    <div class="sidebar">Sidebar</div>
    <div class="main">Main Content</div>
    <div class="footer">Footer</div>
</div>
.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-gap: 10px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

Este código muestra cómo organizar los elementos de una página utilizando CSS Grid.

Posicionamiento Absoluto vs Flexbox vs Grid

El uso de position: absolute todavía tiene sentido en casos muy específicos donde se requiere un control absoluto sobre un elemento, como un menú flotante o un modal. Sin embargo, tanto Flexbox como CSS Grid han demostrado ser más eficientes para layouts generales y ajustables. Además, son más fáciles de manejar cuando se trabaja con diseños responsivos y elementos dinámicos.

Ejemplo de cuándo usar cada técnica

  • Usa position: absolute para elementos como overlays o ventanas modales que deben colocarse en un lugar específico sin influir en el flujo del documento.
  • Usa Flexbox cuando necesites organizar elementos en una fila o columna, como menús, barras de navegación, o para centrar elementos en un contenedor.
  • Usa CSS Grid para layouts complejos, como el diseño general de una página, una galería de imágenes, o cualquier estructura que necesite manejar filas y columnas de manera más controlada.

Conclusión

El posicionamiento en CSS ha evolucionado mucho desde el uso simple de position. Ahora, técnicas como Flexbox y CSS Grid nos permiten crear diseños más complejos y responsivos de manera más eficiente. Es importante dominar estas técnicas para tener más control sobre el diseño de nuestras páginas y adaptarlas a diferentes dispositivos.

CSS Variables (Custom Properties)

Las variables CSS, también conocidas como Custom Properties, permiten almacenar valores y reutilizarlos en tu CSS. Se definen con el prefijo -- y se pueden utilizar mediante la función var().


:root {
    --main-bg-color: #3498db;
    --main-text-color: #ffffff;
}

body {
    background-color: var(--main-bg-color);
    color: var(--main-text-color);
}

En este ejemplo, --main-bg-color y --main-text-color son variables que definen el color de fondo y el color del texto, respectivamente.

Ejercicio

Animaciones y Transiciones

Las animaciones y transiciones permiten crear efectos de cambio suave en los elementos del sitio web. Las transiciones se aplican a cambios de estado, mientras que las animaciones permiten secuencias de movimientos más complejas.

Transiciones

Las transiciones controlan la velocidad de cambio de las propiedades CSS.


.button {
    background-color: #3498db;
    transition: background-color 0.3s;
}

.button:hover {
    background-color: #2ecc71;
}

En este ejemplo, al pasar el cursor sobre el botón, el color de fondo cambiará suavemente.

Animaciones

Las animaciones se definen mediante @keyframes, que describe los cambios en un elemento en diferentes etapas.


@keyframes slideIn {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

.element {
    animation: slideIn 1s ease-in-out;
}

Este código mueve el elemento desde fuera de la pantalla hacia su posición original.

Ejercicio

Crea una animación para un cuadro de texto que cambie de color y posición cuando el usuario haga clic en él.

Responsive Design con Media Queries

El diseño responsive permite que un sitio web se vea bien en diferentes dispositivos y tamaños de pantalla. Se logra utilizando media queries para aplicar estilos específicos según el tamaño de la pantalla.


@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
}

@media (min-width: 601px) {
    .container {
        flex-direction: row;
    }
}

En este ejemplo, los elementos dentro de .container se mostrarán en columna para pantallas pequeñas y en fila para pantallas más grandes.

Ejercicio

CSS Grid Layout Avanzado

CSS Grid Layout es una poderosa herramienta para crear diseños complejos y flexibles. Permite dividir una página en regiones y definir la relación de tamaño, posición y capa entre sus partes.


.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: auto;
    gap: 10px;
}

.header {
    grid-column: 1 / 3;
    grid-row: 1;
}

.sidebar {
    grid-column: 1;
    grid-row: 2;
}

.main-content {
    grid-column: 2;
    grid-row: 2;
}

En este ejemplo, .header ocupa dos columnas, .sidebar ocupa la primera columna, y .main-content ocupa la segunda columna.

CSS Clipping and Masking

CSS Clipping and Masking permiten mostrar solo partes específicas de un elemento, basándose en una forma o una imagen de máscara.

Clipping

Con la propiedad clip-path, puedes crear formas complejas que determinan qué partes de un elemento se mostrarán.


.clipped {
    clip-path: circle(50% at 50% 50%);
}

Este ejemplo crea un clip circular centrado en el elemento.

Masking

La propiedad mask se utiliza para ocultar partes de un elemento, basándose en una imagen de máscara.


.masked {
    mask-image: url('mask.png');
}

Este ejemplo aplica una máscara basada en la imagen mask.png.

CSS Blend Modes

Los modos de mezcla CSS permiten combinar el contenido de los elementos superpuestos de diferentes maneras, creando efectos visuales únicos.


.blended {
    background-image: url('image.jpg');
    background-blend-mode: multiply;
    background-color: rgba(255, 0, 0, 0.5);
}

En este ejemplo, la imagen de fondo y el color de fondo se mezclan usando el modo de mezcla multiply.

Introducción a Bootstrap

Bootstrap es un framework de front-end que facilita el desarrollo de sitios web responsivos y móviles. Proporciona una colección de herramientas CSS y JavaScript predefinidas para crear rápidamente interfaces de usuario modernas y atractivas.

Características Clave

  • Sistema de cuadrícula flexible (Grid system).
  • Componentes de UI como botones, formularios, tarjetas, etc.
  • Responsive design y utilidades para distintos tamaños de pantalla.
  • Extensibilidad mediante plugins de JavaScript.

Ejemplo Básico

Para utilizar Bootstrap, incluye los enlaces a los archivos CSS y JS en tu proyecto:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css">
<script src="https://code.jquery.com/jquery-3.2.1.slim.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js"></script>

Ejemplo de Uso del Sistema de Cuadrícula

<div class="container">
    <div class="row">
        <div class="col-sm">
            Columna 1
        </div>
        <div class="col-sm">
            Columna 2
        </div>
        <div class="col-sm">
            Columna 3
        </div>
    </div>
</div>

Este código crea una disposición de tres columnas que se ajusta según el tamaño de la pantalla.

Ejercicio

Introducción a Sass

Sass (Syntactically Awesome Stylesheets) es un preprocesador CSS que permite usar variables, anidamiento, mixins, funciones y más, para escribir CSS de manera más eficiente y organizada.

Características Clave

  • Variables para almacenar valores como colores, fuentes, etc.
  • Anidamiento de reglas para una estructura más clara.
  • Mixins para reutilizar bloques de código CSS.
  • Funciones y operaciones para manipular valores CSS.

Ejemplo Básico

// Definición de variables
$primary-color: #3498db;
$secondary-color: #2ecc71;

// Uso de variables y anidamiento
body {
    font-family: Arial, sans-serif;
    background-color: $primary-color;
    color: white;

    .header {
        background-color: $secondary-color;
        padding: 10px;
    }
}

// Mixins
@mixin border-radius($radius) {
    -webkit-border-radius: $radius;
    -moz-border-radius: $radius;
    -ms-border-radius: $radius;
    border-radius: $radius;
}

// Uso de mixins
.button {
    @include border-radius(5px);
    background-color: $secondary-color;
}

Este ejemplo muestra cómo usar variables, anidamiento y mixins en Sass.

Ejercicio

Escribe un archivo Sass que defina una paleta de colores, y utilice mixins para crear botones estilizados y secciones con bordes redondeados. Compílalo a CSS y aplícalo a una página web.

Diseño de un Sitio Web

El diseño de un sitio web efectivo requiere planificación y consideración cuidadosa de varios elementos clave para atraer y retener a los usuarios. A continuación se detallan los pasos importantes:

1. Elección de la Paleta de Colores

La paleta de colores debe reflejar la identidad de la marca y el público objetivo. Herramientas como Coolors o Adobe Color pueden ayudar a crear combinaciones armoniosas.

2. Elección de la Tipografía

Selecciona tipografías que sean legibles y apropiadas para el tono del sitio. Google Fonts ofrece una gran variedad de fuentes gratuitas.

3. Diseño del Mockup

El mockup es un prototipo visual del sitio web. Se puede diseñar usando herramientas como Figma, Sketch o Adobe XD. Es útil para visualizar la estructura y disposición antes de empezar a codificar.

4. Implementación

Una vez aprobado el diseño, se procede a la implementación usando HTML, CSS y JavaScript. Frameworks como Bootstrap pueden acelerar el proceso.

5. Recomendaciones para una Landing Page Atrayente

  • Título claro y llamativo.
  • Uso de imágenes de alta calidad y elementos visuales consistentes.
  • Call to Action (CTA) destacado.
  • Navegación sencilla y clara.
  • Optimización para dispositivos móviles.

6. Tipo de Imágenes a Utilizar

Usa imágenes relevantes y de alta calidad. Recurre a bancos de imágenes como Unsplash, Pexels, o Shutterstock, y asegúrate de que las imágenes sean coherentes con el mensaje y estilo del sitio.

Ejercicio Final

Diseña y desarrolla una landing page para un producto o servicio ficticio. Define una paleta de colores, selecciona las tipografías adecuadas, crea un mockup y desarrolla la página usando HTML, CSS (puedes usar Sass) y Bootstrap. Asegúrate de que sea responsive y de alta calidad visual.
