¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se utiliza principalmente para añadir interactividad a las páginas web. Gracias a él, podemos hacer que los elementos cambien, aparezcan mensajes, animaciones, formularios dinámicos y mucho más.

Es un lenguaje interpretado, lo que significa que no es necesario compilarlo: el navegador lo lee y ejecuta directamente.

Aunque su nombre lo parezca, JavaScript no tiene relación con Java. Son lenguajes distintos. El nombre se debe a una decisión comercial.

¿Cómo incluir JavaScript en una página web?

Código dentro del mismo documento HTML

<script type="text/javascript">
  alert("Un mensaje de prueba");
</script>

Código JavaScript en un archivo externo

// archivo llamado codigo.js
alert("Un mensaje de prueba");

<script type="text/javascript" src="/js/codigo.js"></script>

Código JavaScript en los elementos HTML

<p onclick="alert('Un mensaje de prueba')">Haz clic aquí</p>

¿Y si JavaScript está desactivado?

<noscript>
  <p>Este sitio necesita JavaScript para funcionar correctamente.</p>
</noscript>

Sintaxis básica de JavaScript

  • Espacios y saltos de línea: Puedes usarlos para hacer tu código más legible.
  • Mayúsculas y minúsculas: nombre y Nombre son variables distintas.
  • Tipos de datos dinámicos: Las variables pueden cambiar de tipo según el valor.
  • Punto y coma: Aunque no es obligatorio, se recomienda.
  • Comentarios: Explican el código sin ejecutarse.

Programación básica

Antes de comenzar a desarrollar aplicaciones con JavaScript, es fundamental conocer los elementos básicos del lenguaje. Si ya sabes programar en otros lenguajes, esto te servirá para familiarizarte con su sintaxis. Si no, esta sección será tu punto de partida.

Variables

Una variable es un contenedor donde almacenamos datos. Nos permite reutilizar información y realizar operaciones con ella.

¿Cómo se declara una variable?

Usamos la palabra clave let para declarar una variable. Esto le dice al navegador que queremos reservar un espacio en memoria para guardar un valor.

let mensaje = "Hola mundo";

En este ejemplo, mensaje es el nombre de la variable, y "Hola mundo" es el valor que guarda. También podemos declarar una variable y asignarle el valor más adelante:

let edad;
edad = 25;

⚠️ Consejo: Aunque es posible declarar variables sin usar let (por ejemplo, nombre = "Juan"), no es recomendable. Siempre usa let para evitar errores inesperados.

Reglas para nombrar variables

  • El nombre puede incluir letras, números, _ y $, pero no puede comenzar por un número.
  • JavaScript distingue entre mayúsculas y minúsculas: usuario y Usuario son variables distintas.
  • Evita usar palabras reservadas como let, if o function.

Tipos de datos

Las variables en JavaScript pueden almacenar diferentes tipos de valores. Vamos a ver los más comunes:

Números

Sirven para trabajar con datos numéricos, tanto enteros como decimales.

let iva = 21;
let total = 99.95;

Nota: en JavaScript los decimales usan punto, no coma.

Cadenas de texto (strings)

Una cadena es un conjunto de caracteres, como una palabra o una frase. Se escribe entre comillas simples o dobles.

let saludo = "¡Hola!";
let letra = 'a';

Si el texto incluye comillas, puedes usar una de estas estrategias:

let texto1 = "Ella dijo: 'Hola'";
let texto2 = 'Él contestó: "Hola"';

Para incluir símbolos especiales dentro de una cadena, usamos el carácter de escape \:

  • \n → salto de línea
  • \t → tabulación
  • \\ → barra invertida
let mensaje = "Primera línea\nSegunda línea";

Booleanos

Solo pueden almacenar dos valores: true (verdadero) o false (falso). Se usan para tomar decisiones lógicas.

let esMayor = true;
let tienePermiso = false;

Arrays

Un array es como una lista de datos. Puede contener varios elementos dentro de una sola variable.

let dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes"];

Se accede a cada elemento por su posición (empezando desde el 0):

let primero = dias[0]; // "Lunes"
let tercero = dias[2]; // "Miércoles"

También puedes modificar un valor así:

dias[1] = "Martes cambiado";

Operadores en JavaScript

En JavaScript, los operadores nos permiten hacer operaciones con variables y valores. Podemos usarlos para realizar cálculos matemáticos, asignar valores, comparar datos o tomar decisiones.

Operador de asignación (=)

Este operador se utiliza para guardar un valor en una variable. Lo que está a la derecha se asigna a la variable de la izquierda.

let edad = 25;
let nombre = "Laura";

También podemos asignar el valor de una variable a otra:

let numero1 = 5;
let numero2 = numero1; // ahora numero2 también vale 5

Importante: no confundas este operador con el de comparación (==), que veremos más adelante.

Operadores matemáticos

Sirven para realizar operaciones aritméticas básicas.

  • + suma
  • - resta
  • * multiplicación
  • / división
let a = 10;
let b = 4;
let suma = a + b;        // 14
let resta = a - b;       // 6
let producto = a * b;    // 40
let division = a / b;    // 2.5

Operador módulo (%)

Devuelve el resto de una división. Es muy útil, por ejemplo, para saber si un número es par o impar.

let resultado = 10 % 3; // resultado = 1

Operadores combinados con asignación

Permiten actualizar el valor de una variable de forma más breve.

  • += suma y asigna
  • -= resta y asigna
  • *= multiplica y asigna
  • /= divide y asigna
  • %= módulo y asigna
let x = 5;
x += 3; // x ahora vale 8
x *= 2; // x ahora vale 16

Operadores de incremento y decremento

Sirven para aumentar o reducir en una unidad el valor de una variable.

let numero = 5;
numero++; // suma 1 → ahora vale 6
numero--; // resta 1 → ahora vale 5

Pueden usarse antes (++x) o después (x++) de la variable. El orden afecta si se usan dentro de una operación:

let a = 5;
let resultado1 = ++a; // primero suma, luego usa → resultado1 = 6

let b = 5;
let resultado2 = b++; // primero usa, luego suma → resultado2 = 5 (b vale 6)

Operadores lógicos

Se usan para combinar condiciones y obtener valores true o false.

Negación (!)

Invierte el valor lógico: si es true, lo convierte en false y viceversa.

let activo = true;
let inactivo = !activo; // false

AND lógico (&&)

Devuelve true solo si ambas condiciones se cumplen.

let tieneEdad = true;
let tieneDNI = true;
let puedeEntrar = tieneEdad && tieneDNI; // true

OR lógico (||)

Devuelve true si al menos una condición se cumple.

let tieneEntrada = false;
let esInvitado = true;
let accede = tieneEntrada || esInvitado; // true

Operadores de comparación

Sirven para comparar dos valores. El resultado siempre será true o false.

  • == igual que (sin tener en cuenta el tipo)
  • === igual que (comparando también el tipo)
  • != distinto
  • > mayor que
  • >= mayor o igual que
  • < menor que
  • <= menor o igual que
let a = 5;
let b = "5";

a == b      // true (compara solo el valor)
a === b     // false (compara valor y tipo)
a != b      // false
a !== b     // true

Comparación de cadenas

JavaScript compara letra a letra. Ten en cuenta que A < a porque las mayúsculas van antes que las minúsculas.

"hola" == "hola"   // true
"adios" < "hola"   // true
"Zorro" > "oso"    // false

Estructuras de Control

En la mayoría de los programas, no basta con ejecutar instrucciones una detrás de otra. Muchas veces necesitamos tomar decisiones o repetir acciones según una condición. Para eso existen las estructuras de control.

Condicional if

Permite ejecutar un bloque de código solo si se cumple una condición.

if (condición) {
  // instrucciones que se ejecutan si la condición es verdadera
}

Ejemplo:

let edad = 20;
if (edad >= 18) {
  alert("Eres mayor de edad");
}

if...else

Se utiliza para ejecutar un bloque de código si la condición se cumple y otro bloque si no se cumple.

if (condición) {
  // si la condición es verdadera
} else {
  // si la condición es falsa
}

Ejemplo:

let usuario = "";
if (usuario == "") {
  alert("Por favor, introduce tu nombre");
} else {
  alert("¡Bienvenido, " + usuario + "!");
}

if...else if...else

Permite comprobar múltiples condiciones de forma encadenada.

if (condición1) {
  // instrucciones
} else if (condición2) {
  // otras instrucciones
} else {
  // instrucciones por defecto
}

Ejemplo:

let nota = 7;
if (nota < 5) {
  alert("Insuficiente");
} else if (nota < 7) {
  alert("Aprobado");
} else if (nota < 9) {
  alert("Notable");
} else {
  alert("Sobresaliente");
}

Condicional switch

Es útil cuando queremos comparar una misma variable con varios valores posibles. Es más limpio que muchos if...else.

switch (variable) {
  case valor1:
    // instrucciones
    break;
  case valor2:
    // instrucciones
    break;
  default:
    // instrucciones si ningún caso coincide
    break;
}

Ejemplo:

let dia = "lunes";

switch (dia) {
  case "lunes":
    alert("Comienza la semana");
    break;
  case "viernes":
    alert("¡Por fin es viernes!");
    break;
  default:
    alert("Día normal");
    break;
}

Bucle for

Sirve para repetir instrucciones un número determinado de veces. Necesita tres partes: inicialización, condición y actualización.

for (inicio; condición; incremento) {
  // instrucciones
}

Ejemplo:

for (let i = 0; i < 5; i++) {
  console.log("Vuelta número " + i);
}

Bucle for...in

Permite recorrer todos los elementos de un array u objeto.

let frutas = ["manzana", "naranja", "plátano"];

for (let indice in frutas) {
  console.log(frutas[indice]);
}

Bucle while

Repite un bloque de instrucciones mientras se cumpla la condición. Puede ejecutarse 0 o más veces.

while (condición) {
  // instrucciones
}

Ejemplo:

let i = 0;
while (i < 3) {
  console.log("i vale " + i);
  i++;
}

Bucle do...while

Funciona como while, pero garantiza al menos una ejecución del bloque.

do {
  // instrucciones
} while (condición);

Ejemplo:

let respuesta;
do {
  respuesta = prompt("¿Quieres continuar?");
} while (respuesta !== "no");

Funciones en JavaScript

En JavaScript, una función es un bloque de código reutilizable que puede recibir parámetros y opcionalmente devolver un valor. Las funciones permiten modularizar la lógica de un programa.

Declaración de funciones

Se utiliza la palabra clave function, seguida del nombre, una lista de parámetros entre paréntesis y el cuerpo entre llaves.

function saludar(nombre) {
  return "Hola, " + nombre + "!";
}

La función puede invocarse como:

let mensaje = saludar("Lucía");
console.log(mensaje);

Funciones sin valor de retorno

Si no se especifica un return, la función devuelve undefined. Este tipo de funciones se usa para ejecutar instrucciones sin necesidad de devolver resultado.

function mostrarAviso() {
  alert("Acción realizada correctamente.");
}

Parámetros y argumentos

Los parámetros se definen en la declaración. Los valores concretos se pasan como argumentos al llamar a la función.

function calcularArea(base, altura) {
  return (base * altura) / 2;
}

let area = calcularArea(10, 5);

Parámetros con valor por defecto

JavaScript permite asignar valores por defecto a los parámetros desde ES6.

function saludar(nombre = "usuario") {
  console.log("Hola, " + nombre);
}

saludar();        // Hola, usuario
saludar("María"); // Hola, María

Funciones como expresiones

También se pueden definir como expresiones y asignarlas a variables:

const multiplicar = function(a, b) {
  return a * b;
};

Arrow functions

Sintaxis más concisa introducida en ES6:

const saludar = (nombre) => "Hola, " + nombre;

const cuadrado = x => x * x; // Con un solo parámetro y una única expresión

El DOM en JavaScript

El DOM (Document Object Model) es una representación jerárquica de una página web en forma de árbol. JavaScript permite acceder, modificar y manipular cualquier parte del documento HTML a través de este modelo.

Acceso a elementos

JavaScript proporciona varios métodos para acceder a los elementos del documento:

  • getElementById: Accede a un elemento por su atributo id.
  • getElementsByClassName: Devuelve una colección de elementos con una clase determinada.
  • getElementsByTagName: Selecciona todos los elementos con una etiqueta específica.
  • querySelector y querySelectorAll: Permiten seleccionar elementos usando selectores CSS.
let encabezado = document.getElementById("titulo");
let items = document.querySelectorAll(".item");
let parrafos = document.getElementsByTagName("p");

Modificación de contenido

Una vez obtenido el elemento, se pueden modificar sus propiedades:

// Cambiar el texto
encabezado.textContent = "Título actualizado";

// Modificar el contenido HTML interno
encabezado.innerHTML = "<em>Nuevo</em> título";

Modificación de estilos

Se puede acceder a las propiedades CSS mediante .style:

encabezado.style.color = "blue";
encabezado.style.fontSize = "24px";

Gestión de clases

Para añadir, quitar o comprobar clases en un elemento, se utiliza classList:

encabezado.classList.add("resaltado");
encabezado.classList.remove("resaltado");
encabezado.classList.toggle("activo");

Creación y eliminación de elementos

También se pueden crear elementos nuevos o eliminar existentes:

let nuevoParrafo = document.createElement("p");
nuevoParrafo.textContent = "Este es un párrafo nuevo";

document.body.appendChild(nuevoParrafo);

document.body.removeChild(nuevoParrafo);

Eventos en JavaScript

En JavaScript, los eventos son mecanismos que permiten reaccionar a las acciones del usuario o del navegador (clics, movimientos del ratón, carga de la página, pulsaciones de teclas, etc.).

¿Qué es un Event Listener?

Un event listener es una función que se ejecuta automáticamente cuando ocurre un evento específico en un elemento HTML. En JavaScript, el método addEventListener() se usa para asociar un evento con una función que lo gestionará.

Su estructura básica es:

elemento.addEventListener("evento", funcionManejadora);

Ejemplo práctico:

let boton = document.getElementById("miBoton");

boton.addEventListener("click", function() {
  alert("Has hecho clic en el botón");
});

En este caso:

  • miBoton es el ID del elemento HTML (por ejemplo, un <button>).
  • click es el nombre del evento que queremos escuchar.
  • function() { ... } es la función manejadora que se ejecuta cuando ocurre el evento.

Eventos más utilizados

A continuación, se presenta una selección de eventos comunes en JavaScript, con ejemplos de uso para cada uno:

  • click: se activa al hacer clic en un elemento.
  • dblclick: se activa al hacer doble clic.
  • mouseover: se activa cuando el puntero entra en el elemento.
  • mouseout: se activa cuando el puntero sale del elemento.
  • keydown: se activa al pulsar una tecla.
  • keyup: se activa al soltar una tecla.
  • change: se activa cuando cambia el valor de un input, select, etc.
  • submit: se activa al enviar un formulario.
  • load: se activa cuando la página o imagen ha cargado completamente.
// Cambiar color de fondo al pasar el ratón
document.getElementById("caja").addEventListener("mouseover", function() {
  this.style.backgroundColor = "lightblue";
});
// Detectar una tecla pulsada
document.addEventListener("keydown", function(evento) {
  console.log("Tecla pulsada: " + evento.key);
});
// Validar formulario al enviarlo
document.getElementById("formulario").addEventListener("submit", function(e) {
  e.preventDefault(); // Evita el envío real
  alert("Formulario enviado correctamente (simulado).");
});

Funciones manejadoras nombradas

Es una buena práctica definir funciones manejadoras con nombre, especialmente si se reutilizan o se necesitan eliminar más tarde.

function saludarUsuario() {
  alert("¡Bienvenido!");
}

document.getElementById("botonSaludo").addEventListener("click", saludarUsuario);
