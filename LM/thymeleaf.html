Introducción a Thymeleaf

Thymeleaf es un motor de plantillas para Java que permite generar páginas HTML dinámicas en aplicaciones web basadas en Spring Boot.

Configuración de Thymeleaf en Spring Boot

Para utilizar Thymeleaf en Spring Boot, añade la siguiente dependencia en pom.xml:


<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>

Uso básico de Thymeleaf

Comparación entre HTML estático y Thymeleaf

Aquí tienes un ejemplo de una página HTML estática:


<h1>Bienvenido a nuestra tienda</h1>
<p>Usuario: Juan Pérez</p>
<p>Saldo disponible: $100</p>

Ahora veamos cómo se convierte en Thymeleaf:


<h1>Bienvenido a nuestra tienda</h1>
<p th:text="'Usuario: ' + ${usuario}">Usuario</p>
<p th:text="'Saldo disponible: $' + ${saldo}">Saldo</p>

Si en el controlador tenemos:


@GetMapping("/")
public String home(Model model) {
    model.addAttribute("usuario", "Juan Pérez");
    model.addAttribute("saldo", 100);
    return "index";
}

Variables en Thymeleaf

Uso de Variables

Las variables en Thymeleaf permiten mostrar datos dinámicos en la plantilla:


<p th:text="${nombre}">Nombre del usuario</p>

Propiedades de Objetos

Podemos acceder a atributos de objetos enviados desde el controlador:


<p th:text="${usuario.nombre}">Nombre</p>
<p th:text="${usuario.email}">Email</p>

Condiciones en Thymeleaf

Evaluación de condiciones con th:if

Podemos mostrar o esconder elementos según condiciones:


<p th:if="${saldo > 0}">Saldo disponible</p>
<p th:unless="${saldo > 0}">Sin saldo disponible</p>

Uso de th:switch


<div th:switch="${rol}">
    <p th:case="'admin'">Eres administrador</p>
    <p th:case="'user'">Eres usuario</p>
    <p th:case="*">Rol desconocido</p>
</div>

Iteraciones en Thymeleaf

Uso de th:each para recorrer listas


<ul>
    <li th:each="producto : ${productos}" th:text="${producto}">Ejemplo de producto</li>
</ul>

Índices y contadores en bucles


<ul>
    <li th:each="producto, stat : ${productos}">
        Producto #[[${stat.index}]]: [[${producto}]]
    </li>
</ul>

Otros conceptos importantes en Thymeleaf

Operadores

  • + (concatenación): 'Hola ' + ${nombre}
  • Comparación: ${precio > 100}
  • Lógicos: ${activo && saldo > 0}
