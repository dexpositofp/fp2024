Presentación

Bienvenidos al módulo de Lenguaje de Marcas. Este curso cubre los fundamentos de los lenguajes de marcado como HTML y CSS, que son esenciales para el desarrollo web.

Objetivos del Curso

  • Comprender la estructura y el propósito de HTML y CSS.
  • Aprender a utilizar las etiquetas y propiedades para crear y diseñar páginas web.
  • Desarrollar habilidades prácticas a través de ejercicios y proyectos.

Materiales y Recursos

  • Editor de código (Visual Studio Code, Sublime Text).
  • Navegadores web para pruebas.
  • Recursos en línea como MDN Web Docs y W3Schools.

Estructura del Curso

El curso está dividido en las siguientes unidades:

  • Lenguajes de marcas: Conceptos básicos y la importancia del lenguaje de marcas.
  • HTML: Historia, etiquetas básicas, y estructura de documentos HTML.
  • CSS: Selección de elementos, propiedades de estilo, y diseño responsivo.
  • JavaScript: Fundamentos de la programación y cómo añadir interactividad a las páginas web.

Evaluación

La evaluación de cada RA se basa en:

  • Participación en clase y ejercicios prácticos.
  • Prueba objetiva individual (60% de la nota de la unidad).
  • Proyecto final (40% de la nota de la unidad).

Importante: Para obtener la nota del proyecto, es imprescindible superar una prueba de validación individual. La nota mínima para los exámenes es de 5. Si no se supera la prueba de validación, la calificación del proyecto será un 0.
