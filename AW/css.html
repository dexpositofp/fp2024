Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of an HTML document.

With CSS, we can control the layout, arrangement, and appearance of elements on a web page.

CSS Syntax

The syntax of CSS consists of selectors and declarations.

A selector targets the HTML element you want to style. A declaration is a property and its value.


/* Example of CSS syntax */
selector {
    property: value;
}

Selectores in CSS

CSS offers a variety of selectors to target the HTML elements you want to style:

  • Element Selector: Selects all elements of a specific type. Example: p { color: red; }
  • Class Selector: Selects elements with a specific class. Example: .class { color: blue; }
  • ID Selector: Selects an element with a specific ID. Example: #id { color: green; }
  • Attribute Selector: Selects elements with a specific attribute. Example: [type="text"] { color: purple; }
  • Pseudo-Class Selector: Selects elements in a specific state. Example: a:hover { color: orange; }
  • Pseudo-Element Selector: Selects a specific part of an element. Example: p::first-line { color: pink; }

CSS Properties

Font-Related Properties

  • font-family: Defines the font family. font-family: 'Arial', sans-serif;
  • font-size: Defines the font size. font-size: 16px;
  • font-weight: Defines the font thickness. font-weight: bold;
  • font-style: Defines the font style (normal, italic, etc.). font-style: italic;
  • line-height: Defines the line height. Example: line-height: 1.5;
  • text-align: Aligns the text. text-align: center;
  • text-decoration: Defines text decoration (underline, line-through, etc.). text-decoration: underline;
  • text-transform: Controls text capitalization. text-transform: uppercase;

Background-Related Properties

  • background-color: Defines the background color. background-color: lightblue;
  • background-image: Defines a background image. background-image: url('image.jpg');
  • background-repeat: Defines how the background image repeats. background-repeat: no-repeat;
  • background-size: Defines the size of the background image. Example: background-size: cover;
  • background-position: Defines the position of the background image. Example: background-position: center;

Linking HTML and CSS

To apply CSS styles to an HTML document, we need to link both files. There are several ways to do this:

1. Using the <link> Tag in the <head>

The most common and recommended way is to include an external stylesheet using the <link> tag in the <head> section of your HTML document. This helps keep HTML and CSS separate, making it easier to maintain and manage styles.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Welcome to my web page</h1>
    <p>This is an example of how to link HTML and CSS.</p>
</body>
</html>

2. Using the <style> Tag in the <head>

Another way to add CSS styles is by using the <style> tag directly in the <head> of the HTML document. This technique is useful for applying quick or specific styles that do not justify creating a separate CSS file.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
        }
        h1 {
            color: #333;
        }
    </style>
</head>
<body>
    <h1>Welcome to my web page</h1>
    <p>This is an example of how to use the <style> tag for CSS in HTML.</p>
</body>
</html>

3. Using Inline Styles

You can also apply CSS styles directly to HTML elements using the <style> attribute. This practice is known as inline styles and is not recommended for large projects due to the difficulty of maintaining and modifying styles.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <h1 style="color: blue;">Welcome to my web page</h1>
    <p style="font-size: 16px;">Example of inline styles in HTML.</p>
</body>
</html>

Box Model

The box model is a fundamental concept in CSS. It describes how elements are structured on the web page, allowing for the manipulation of margins, borders, padding, and content.

The box model consists of four parts:

  • Content: the area where the content (text, images, etc.) is displayed.
  • Padding: the space between the content and the border.
  • Border: the outline of the box.
  • Margin: the space between the box and other elements.
.element {
    width: 200px;
    padding: 10px;
    border: 5px solid #333;
    margin: 20px;
}

In the example above, we have a box with a width of 200px, padding of 10px, a border of 5px, and a margin of 20px. These values determine the size and spacing of the box.

Borders and Their Properties

Border properties in CSS allow you to add an outline to elements. Some main border properties include:

  • border-width: sets the border thickness.
  • border-style: sets the border style (solid, dotted, dashed, etc.).
  • border-color: sets the border color.
.border-element {
    border-width: 4px;
    border-style: solid;
    border-color: #007bff;
}

You can also use border-radius to round the borders, as shown in the following example:

.rounded-border-element {
    border: 2px solid #007bff;
    border-radius: 10px;
}

CSS Positioning

CSS positioning is fundamental for controlling the layout of a webpage. It defines how and where elements should be placed within the available space on the page. Over time, positioning techniques have evolved from using position (absolute, relative, fixed, and static) to modern techniques like Flexbox and CSS Grid, which greatly simplify working with responsive layouts.

Types of CSS Positioning

  • static: This is the default value. Elements are positioned in the natural document flow.
  • relative: Moves the element from its original position in the document flow without changing the space it occupies.
  • absolute: Positions an element relative to its closest container with position: relative or absolute.
  • fixed: Fixes the element at a specific position on the screen, regardless of scroll.
  • sticky: A combination of relative and fixed. The element behaves as relative until it reaches a specific position, then stays fixed.

Basic Positioning Examples:

position: relative moves the element relative to its natural position:


.relative-element {
  position: relative;
  top: 10px; /* The element moves 10px down */
  left: 20px; /* The element moves 20px to the right */
}

position: absolute removes the element from the normal document flow:


.absolute-element {
  position: absolute;
  top: 50px;  /* Positioned 50px from the top of its container */
  left: 30px; /* Positioned 30px from the left of its container */
}

For an element with position: absolute to work, it must be within a container that has position: relative or absolute.

Flexbox

The flexible box model, or Flexbox, is ideal for one-dimensional layouts (either in a row or column). It is very useful for distributing space among items and aligning them within a container.

Key Flexbox Concepts

  • justify-content: Controls the alignment of items along the main axis (horizontal or vertical).
  • align-items: Aligns items along the cross axis (perpendicular).
  • flex-direction: Defines whether items are aligned in a row (row) or column (column).
  • flex-wrap: Controls whether items should remain in a single row or wrap onto multiple rows.

Basic Flexbox example:


.container {
  display: flex;
  justify-content: space-between; /* Equal spacing between items */
  align-items: center; /* Vertically centers items */
  flex-wrap: wrap; /* Allows items to wrap to the next line if needed */
}

This code creates a flexible layout where the items inside the .container will align with space between them and center vertically.

Flexible Distribution with flex-grow and flex-shrink

Using flex-grow and flex-shrink allows you to control how much an item should grow or shrink within a flex container. For example:


.item {
  flex-grow: 1; /* This item will grow to take up more available space */
  flex-shrink: 2; /* This item will shrink twice as fast as others when the container is smaller */
}

What is CSS Grid?

CSS Grid is a two-dimensional layout system that divides a page into cells or areas, allowing for precise placement of elements. Unlike Flexbox, which is one-dimensional (rows or columns), CSS Grid allows for working with both dimensions at once.

Key CSS Grid Properties

1. Grid Container

To activate Grid mode on a container, use the property display: grid;:

.grid-container {
    display: grid;
}

2. Defining Columns and Rows

The layout of columns and rows is mainly controlled with the properties grid-template-columns and grid-template-rows. Hereâ€™s a simple example:

.grid-container {
    display: grid;
    grid-template-columns: 200px 200px;
    grid-template-rows: 100px 100px;
}

This example creates a grid with two columns of 200px and two rows of 100px.

3. Cell Spacing

To add space between grid cells, use grid-gap:

.grid-container {
    display: grid;
    grid-template-columns: 200px 200px;
    grid-template-rows: 100px 100px;
    grid-gap: 10px;
}

4. Grid Areas

Grid areas allow defining blocks of cells where content can be placed. They are used with the property grid-template-areas:

.grid-container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
}

5. Element Placement

Placement is controlled by properties like grid-column and grid-row, which determine in which column and row each element is placed:

.item {
    grid-column: 1 / 3;
    grid-row: 2 / 4;
}

6. Measurement Units

Units like fr (fraction), px, %, and auto provide flexibility in distributing elements in the grid.

.grid-container {
    grid-template-columns: 1fr 2fr 1fr;
}

In this case, the columns are set to a 1:2:1 ratio.

Complete Example

Here is a full example of a CSS Grid layout with a container and some elements:

<div class="grid-container">
    <div class="header">Header</div>
    <div class="sidebar">Sidebar</div>
    <div class="main">Main Content</div>
    <div class="footer">Footer</div>
</div>
.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-gap: 10px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

This code shows how to organize page elements using CSS Grid.

Absolute Positioning vs Flexbox and CSS Grid

Absolute positioning is ideal for elements that need to stay fixed relative to another element or the viewport, while Flexbox and CSS Grid are better for fluid layouts where space distribution and element alignment are prioritized.

Pseudo-Classes in CSS

CSS pseudo-classes are keywords added to selectors to apply styles for a specific state of an element, such as when a user hovers over it or a link has been visited.

:hover

The :hover pseudo-class is activated when the user hovers over an element. It's useful for links and buttons.

Hover over me!

.btn-hover {
    padding: 10px 20px;
    color: white;
    background-color: #3498db;
    text-decoration: none;
    border-radius: 5px;
    transition: background-color 0.3s;
}

.btn-hover:hover {
    background-color: #2980b9;
}

:active

The :active pseudo-class applies to an element while it is being pressed, ideal for buttons.

Press me!

.btn-active {
    padding: 10px 20px;
    color: white;
    background-color: #e74c3c;
    text-decoration: none;
    border-radius: 5px;
    transition: transform 0.1s;
}

.btn-active:active {
    background-color: #c0392b;
    transform: scale(0.95);
}

:focus

The :focus pseudo-class applies styles to an element when it receives keyboard focus or a click.



.input-focus {
    padding: 8px;
    border: 1px solid #ccc;
    outline: none;
    transition: border-color 0.3s;
}

.input-focus:focus {
    border-color: #3498db;
}

Contextual Definitions

Contextual definitions allow you to apply styles based on the position of elements within the HTML structure.

  • Descendant Selector: Selects elements within a specific container. Ex: div p { color: blue; }
  • Child Selector: Selects elements that are direct children of a specific container. Ex: div > p { color: green; }
  • Adjacent Sibling Selector: Selects an element that is the immediate next sibling of another. Ex: h1 + p { margin-top: 0; }
  • General Sibling Selector: Selects all sibling elements after a preceding element. Ex: h1 ~ p { color: red; }

Direct Child Selector >

This selector selects only the direct children of an element. It allows you to apply styles only to elements at an immediate level within another.

/* Applies style only to paragraphs that are direct children of a div */
div > p {
    margin-bottom: 10px;
}

In this case, only <p> elements directly within a <div> will be styled.

Adjacent Sibling Selector +

This selector applies styles to the first element that immediately follows a specified element.

/* Applies style to the paragraph that follows an h2 */
h2 + p {
    color: #555;
    font-style: italic;
}

In this case, only the first <p> immediately following an <h2> will be affected.

General Sibling Selector ~

This selector affects all sibling elements that share the same parent and come after a specified element.

/* Style applied to all paragraphs following an h2 */
h2 ~ p {
    color: #007bff;
}

Unlike +, this selector applies styles to all sibling elements instead of just the immediate one.
