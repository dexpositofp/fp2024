Introduction to CSS

CSS (Cascading Style Sheets) is a language used to describe the presentation of an HTML document.

With CSS, we can control the layout, arrangement, and appearance of elements on a web page.

CSS Syntax

The syntax of CSS consists of selectors and declarations.

A selector targets the HTML element you want to style. A declaration is a property and its value.


/* Example of CSS syntax */
selector {
    property: value;
}

Selectores in CSS

CSS offers a variety of selectors to target the HTML elements you want to style:

  • Element Selector: Selects all elements of a specific type. Example: p { color: red; }
  • Class Selector: Selects elements with a specific class. Example: .class { color: blue; }
  • ID Selector: Selects an element with a specific ID. Example: #id { color: green; }
  • Attribute Selector: Selects elements with a specific attribute. Example: [type="text"] { color: purple; }
  • Pseudo-Class Selector: Selects elements in a specific state. Example: a:hover { color: orange; }
  • Pseudo-Element Selector: Selects a specific part of an element. Example: p::first-line { color: pink; }

CSS Properties

Font-Related Properties

  • font-family: Defines the font family. font-family: 'Arial', sans-serif;
  • font-size: Defines the font size. font-size: 16px;
  • font-weight: Defines the font thickness. font-weight: bold;
  • font-style: Defines the font style (normal, italic, etc.). font-style: italic;
  • line-height: Defines the line height. Example: line-height: 1.5;
  • text-align: Aligns the text. text-align: center;
  • text-decoration: Defines text decoration (underline, line-through, etc.). text-decoration: underline;
  • text-transform: Controls text capitalization. text-transform: uppercase;

Background-Related Properties

  • background-color: Defines the background color. background-color: lightblue;
  • background-image: Defines a background image. background-image: url('image.jpg');
  • background-repeat: Defines how the background image repeats. background-repeat: no-repeat;
  • background-size: Defines the size of the background image. Example: background-size: cover;
  • background-position: Defines the position of the background image. Example: background-position: center;

Linking HTML and CSS

To apply CSS styles to an HTML document, we need to link both files. There are several ways to do this:

1. Using the <link> Tag in the <head>

The most common and recommended way is to include an external stylesheet using the <link> tag in the <head> section of your HTML document. This helps keep HTML and CSS separate, making it easier to maintain and manage styles.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Welcome to my web page</h1>
    <p>This is an example of how to link HTML and CSS.</p>
</body>
</html>

2. Using the <style> Tag in the <head>

Another way to add CSS styles is by using the <style> tag directly in the <head> of the HTML document. This technique is useful for applying quick or specific styles that do not justify creating a separate CSS file.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
        }
        h1 {
            color: #333;
        }
    </style>
</head>
<body>
    <h1>Welcome to my web page</h1>
    <p>This is an example of how to use the <style> tag for CSS in HTML.</p>
</body>
</html>

3. Using Inline Styles

You can also apply CSS styles directly to HTML elements using the <style> attribute. This practice is known as inline styles and is not recommended for large projects due to the difficulty of maintaining and modifying styles.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <h1 style="color: blue;">Welcome to my web page</h1>
    <p style="font-size: 16px;">Example of inline styles in HTML.</p>
</body>
</html>

The CSS Box Model

The box model is a fundamental concept in CSS that describes how HTML elements are represented on the web page.

Each element is a box with margins, borders, padding, and content.


/* Example of box model */
element {
    width: 100px;
    height: 100px;
    margin: 10px;
    padding: 20px;
    border: 1px solid black;
}

Positioning in CSS

Positioning in CSS is crucial for controlling how elements are placed on a web page. There are several positioning methods, each with its own use and behavior.

1. Static Positioning

Static positioning is the default value for all elements.

Elements with position: static are placed in the normal flow of the document, and the top, right, bottom, and left properties have no effect.


.static-box {
    position: static;
    width: 200px;
    height: 100px;
    background-color: lightblue;
}

Tip: Use static positioning when you do not need to adjust the position of the element specifically. It is useful for most elements in the normal flow of the document.

2. Relative Positioning

With position: relative, the element is positioned relative to its original position in the document flow. The top, right, bottom, and left properties move the element from its original position.


.relative-box {
    position: relative;
    top: 20px;
    left: 30px;
    width: 200px;
    height: 100px;
    background-color: lightcoral;
}

Tip: Use relative positioning to slightly adjust the position of an element without removing it from the normal document flow. It is useful when you need a slight offset from its original location.

3. Absolute Positioning

position: absolute removes the element from the normal document flow and positions it relative to the nearest parent element with a positioning value other than static. If no such element exists, it is positioned relative to the document.


.absolute-container {
    position: relative;
    width: 300px;
    height: 200px;
    background-color: lightgrey;
}

.absolute-box {
    position: absolute;
    top: 10px;
    right: 10px;
    width: 150px;
    height: 80px;
    background-color: lightgreen;
}

Tip: Absolute positioning is ideal for placing elements in a specific location within a container. Use it for floating or modal elements that need to be in a fixed position within their parent container.

4. Fixed Positioning

position: fixed positions the element relative to the browser window. The element stays in the same position even when the user scrolls the page.


.fixed-box {
    position: fixed;
    bottom: 20px;
    right: 20px;
    width: 150px;
    height: 80px;
    background-color: lightpink;
}

Tip: Use fixed positioning for elements that need to remain visible at all times, such as navigation buttons or toolbars, regardless of the page scroll.

5. Sticky Positioning

position: sticky is a combination of relative and fixed. The element behaves like relative until it scrolls to a specified position in the viewport, at which point it behaves like fixed.


.sticky-box {
    position: sticky;
    top: 0;
    width: 100%;
    height: 50px;
    background-color: lightgoldenrodyellow;
}

Tip: Sticky positioning is useful for elements like section headers that need to remain visible while scrolling through a long section, but should not be fixed to the screen at all times.

Pseudo-Classes in CSS

Pseudo-classes are used to define the special state of an element.

  • hover: Activated when the user hovers over an element. E.g.: a:hover { color: red; }
  • focus: Activated when an element receives focus. E.g.: input:focus { border-color: blue; }
  • nth-child(n): Selects the nth child of its parent. E.g.: li:nth-child(2) { color: green; }
  • first-child: Selects the first child of its parent. E.g.: p:first-child { font-weight: bold; }
  • last-child: Selects the last child of its parent. E.g.: p:last-child { font-weight: bold; }
  • not(selector): Selects all elements that do not match the given selector. E.g.: input:not([type="text"]) { display: none; }

Context-Based Definitions

Context-based definitions allow applying styles based on the position of elements within the HTML structure.

  • Descendant Selector: Selects elements within a specific container. E.g.: div p { color: blue; }
  • Child Selector: Selects elements that are direct children of a specific container. E.g.: div > p { color: green; }
  • Adjacent Sibling Selector: Selects an element that is the immediate next sibling of another. E.g.: h1 + p { margin-top: 0; }
  • General Sibling Selector: Selects all sibling elements following a previous element. E.g.: h1 ~ p { color: red; }

Flexbox in CSS

Flexbox, or "Flexible Box Layout", is a CSS layout system that allows for creating responsive and aligned interfaces efficiently. It is especially useful for distributing space between elements and aligning them within containers, even when the size of the elements is dynamic or unknown.

How Flexbox Works

The Flexbox system consists of two main components:

  • Flex Container: The parent element that contains the flexible items.
  • Flex Items: The child elements that will be aligned and distributed according to Flexbox rules.

Main Flexbox Properties

Flexbox has several key properties to control the distribution and alignment of flexible items:

  • display: flex; or display: inline-flex; Turns the container into a flex container.
  • flex-direction: Defines the direction in which items are distributed.
  • justify-content: Controls alignment of items along the main axis.
  • align-items: Controls alignment of items along the cross axis.
  • flex-wrap: Allows items to wrap onto multiple lines if they do not fit in one.
  • align-content: Controls the alignment of multiple lines of flex items if there is extra space.

Practical Example of Flexbox

Here is an example of how to apply Flexbox in a container with several items:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Flexbox Example</title>
    <style>
        .container {
            display: flex;
            flex-direction: row;
            justify-content: space-around;
            align-items: center;
            flex-wrap: wrap;
            height: 300px;
            border: 2px solid #3498db;
        }
        .item {
            background-color: #e74c3c;
            color: white;
            padding: 20px;
            margin: 10px;
            font-size: 18px;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="item">Item 1</div>
        <div class="item">Item 2</div>
        <div class="item">Item 3</div>
        <div class="item">Item 4</div>
    </div>
</body>
</html>

CSS Grid Layout

CSS Grid Layout is a CSS layout system that allows for creating complex and responsive structures in a much more intuitive way. While Flexbox focuses on the distribution of elements along a single axis (horizontal or vertical), Grid allows for working in two dimensions (rows and columns) simultaneously.

How Grid Layout Works

The Grid Layout system is based on a grid container that organizes its content into a grid of rows and columns. The child elements within this container are called grid items, which can be positioned in specific locations within the grid.

Main Grid Layout Properties

Grid Layout has several key properties for controlling the structure and distribution of elements:

  • display: grid; or display: inline-grid; Turns a container into a grid container.
  • grid-template-columns: Defines the columns in the grid.
  • grid-template-rows: Defines the rows in the grid.
  • grid-gap: Sets the space between rows and columns.
  • grid-template-areas: Allows organizing specific areas of the grid using names.
  • justify-items: Controls the alignment of items within their cells along the horizontal axis.
  • align-items: Controls the alignment of items within their cells along the vertical axis.

Practical Example of Grid Layout

Here is a basic example of how to apply CSS Grid to organize a simple layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Grid Layout Example</title>
    <style>
        .grid-container {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: auto;
            grid-gap: 20px;
            padding: 20px;
            border: 2px solid #2ecc71;
        }
        .grid-item {
            background-color: #9b59b6;
            color: white;
            padding: 20px;
            text-align: center;
            font-size: 18px;
        }
    </style>
</head>
<body>
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>
</body>
</html>
