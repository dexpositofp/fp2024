Introduction

Welcome to the Web Applications course. In this course, we will explore the fundamentals and applications of markup languages, focusing on HTML and CSS. These languages are the foundation of web development and are essential for creating modern and attractive websites.

We will also work with Content Management Systems, building a complete website or an online store.

Last but not least, we will learn how to program small applications.

Course Objectives

  • Understand the structure and purpose of HTML.
  • Learn to use HTML tags to create structured content.
  • Introduce basic CSS concepts for design and presentation.
  • Learn to install, configure, and modify content management systems.
  • Develop practical skills through exercises and projects.

Materials and Resources

To make the most of this course, we recommend having access to the following tools and resources:

  • A code editor.
  • Various web browsers (Chrome, Firefox, etc.) to test the code.
  • Virtual machines and containers.
  • Online resources, such as MDN Web Docs and W3Schools.

Course Structure

The course is divided into several blocks, each focusing on different aspects of web development:

  • Internet basics: Basic concepts and the importance of web applications.
  • HTML: History, basic tags, and HTML document structure.
  • CSS: Element selection, style properties, and responsive design.
  • CMS: Content management systems.
  • Python: Programming fundamentals and how to add interactivity to web pages.

Assessment

Assessment for the course will be based on five key results of learning:

  1. Install content management systems, identifying their applications and configuring them according to requirements.
  2. Install distance learning management systems, describing the site's structure and directory hierarchy.
  3. Install web file management services, identifying their applications and verifying their integrity.
  4. Install web office applications, describing their features and usage environments.
  5. Install web desktop applications, describing their features and usage environments.

The course is divided into six blocks, each covering a different aspect of the subject:

  • BLOQUE I: Introduction to the Internet (5% of final grade)
  • BLOQUE II: Internet Languages (25% of final grade)
  • BLOQUE III: Content Management Systems (41% of final grade)
  • BLOQUE IV: Web Programming Applications (19% of final grade)
  • BLOQUE V: Web File Management Services (5% of final grade)
  • BLOQUE VI: Distance Learning Management Systems (5% of final grade)

For more information, please refer to the course guide.
