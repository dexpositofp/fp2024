What is a variable?

A variable is like a labeled box. You can put information inside ‚Äî like a name, color, or number ‚Äî and reuse it later in your program.

Example:

Here, the variable name stores the text "Alice", which is then used in the message.

Conditions

A condition lets a program make decisions. Just like people say "If it's raining, take an umbrella", programs also say: "If something is true, do this".

A condition checks whether something is True or False. We use the if statement to run code only when a condition is true.

Simple condition

Only runs when the condition is true:


is_raining = True

if is_raining:
    print("Don't forget an umbrella!")
print("See you soon.")

Explanation: Only the indented line after if runs when the condition is true. Indentation is very important!

Using input with a condition


positive_num = int(input("Enter a positive number: "))

if positive_num < 0:
    print("Negative input set to 0")
    positive_num = 0

print("Final value:", positive_num)

If - else structure

What if you want to do something when the condition is false? Use else.


distance = 0.7

if distance < 1:
    print("Let's walk!")
else:
    print("Let's drive.")

Only one of the two blocks will run.

Nested if statements

We can also place if statements inside others. This is called nesting and helps us organize decisions.


if game == 1:
    if players < 2:
        print("Not enough players")
    elif players > 4:
        print("Too many players")
    else:
        print("Ready to start")

elif game == 2:
    if players < 3:
        print("Not enough players")
    elif players > 6:
        print("Too many players")
    else:
        print("Ready to start")

Using elif helps avoid repetition and makes code easier to read.

Loops

A loop is a block of code that runs repeatedly while a condition is true. It helps automate tasks that need to happen multiple times.

Examples:

  • Check if a phone is idle and then lock it
  • Print all names in a student list
  • Count numbers, repeat messages, etc.

Types of loops in Python:

  • while loop ‚Üí Repeats while a condition is true
  • for loop ‚Üí Repeats for each item in a list or range

While loop

A while loop runs a block of code as long as the condition is True.


count = 1

while count <= 3:
    print("Count:", count)
    count += 1

Output:

  • Count: 1
  • Count: 2
  • Count: 3

Counting with a step:

We can count and check for even numbers using a step:


i = 1

while i <= 20:
    if i % 2 == 0:
        print(i)
    i += 1

For loop

A for loop goes through each item in a sequence like a list or a range.


names = ["Alice", "Bob", "Charlie"]

for name in names:
    print("Hello,", name)

Using range() to count

range() generates a sequence of numbers:

  • range(end) ‚Üí 0 to end (exclusive)
  • range(start, end) ‚Üí start to end (exclusive)
  • range(start, end, step) ‚Üí step size included

for i in range(4):
    print(i)
# Output: 0, 1, 2, 3

for i in range(2, 6):
    print(i)
# Output: 2, 3, 4, 5

for i in range(1, 10, 2):
    print(i)
# Output: 1, 3, 5, 7, 9

Break statement

break stops the loop early if a condition is met.


user_string = "This is a string."

for i in range(len(user_string)):
    if user_string[i] == 'a':
        print("Found at index:", i)
        break

Continue statement

continue skips the current loop and goes to the next iteration.


for i in range(6):
    if i == 3:
        continue
    print(i)

Functions

Functions are the next step toward writing clean, reusable, and organized code. If you find yourself repeating the same instructions, you can turn them into a function!

A function is a named block of code that performs a task. You define it once, and you can use (or call) it many times.

üõ† Built-in functions

Python already has many functions you can use:

  • print() ‚Üí Shows something on screen
  • input() ‚Üí Asks the user for information
  • len() ‚Üí Tells you the length of a string
  • type() ‚Üí Tells you the type of data

name = input("What is your name? ")
print("Hello,", name)
print("Your name has", len(name), "letters")

Defining your own function

You create a function using def followed by the function name and (). The code block must be indented.


def greet():
    """Greets the user."""
    print("Hi there!")
    print("Welcome to Python")

greet()
greet()

This function prints two lines every time it is called.

Parameters

You can pass values to your function using parameters:


def greet(name):
    print("Hello,", name)

greet("Ana")
greet("Javi")

Returning values

Functions can also return results for you to use later.


def add(a, b):
    return a + b

total = add(5, 3)
print("Total:", total)

Full function structure

A complete function usually looks like this:


def function_name(parameters):
    """This is a docstring: explains what the function does."""
    # statements
    return something

You must define a function before using it.

Why use functions?

  • Modularity: Group related code together
  • Reusability: Call the function as many times as needed
  • Teamwork: Share functions with others
  • Maintainability: Easier to update one function than fix many places

üîÄ Control flow and functions

Control flow is the path the program follows. Normally, Python reads code line by line.

When Python finds a function call, it jumps to the function, runs it, and then returns to where it left off.


def show_message():
    print("Hello from inside the function!")

print("Start")
show_message()
print("End")

Output:

  • Start
  • Hello from inside the function!
  • End
