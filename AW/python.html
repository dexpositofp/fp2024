Introduction to Python

Python is a high-level programming language designed to be easy to read and simple to write. It is widely used in different fields, including web development, artificial intelligence, data science, and automation.

Why Use Python?

  • Simple and Readable: Uses clear and straightforward syntax.
  • Cross-Platform: Runs on Windows, Mac, and Linux without modifications.
  • Extensive Libraries: Thousands of prebuilt libraries simplify coding.
  • Community Support: Large global community provides learning resources.

Basic Syntax and Indentation

Python does not use curly braces ({ }) like other languages. Instead, it relies on indentation to define code blocks.

Example:


if True:
    print("This is inside the block")
print("This is outside the block")

Note: Proper indentation is crucial in Python. Incorrect indentation will cause an error.

Data Types in Python

Python supports various data types:

  • Integers (int): Whole numbers (e.g., 5, -3, 100)
  • Floating-point (float): Decimal numbers (e.g., 3.14, -0.01, 2.0)
  • Strings (str): Text data (e.g., "Hello, world!")
  • Lists (list): Ordered collection (e.g., [1, 2, 3])
  • Tuples (tuple): Immutable collection (e.g., (1, 2, 3))
  • Dictionaries (dict): Key-value pairs (e.g., {"name": "Alice", "age": 25})

Conditional Statements

Python uses if, elif, and else to control the flow of execution.

Example:


x = 10

if x > 5:
    print("x is greater than 5")
elif x == 5:
    print("x is exactly 5")
else:
    print("x is less than 5")

Types of Conditions:

  • Comparison:==, !=, >,<
  • Logical: and, or, not

Loops in Python

Loops allow repeated execution of code blocks.

For Loop

Used when the number of iterations is known.


for i in range(5):
    print(i)  # Outputs 0, 1, 2, 3, 4

While Loop

Used when iterations depend on a condition.


x = 0
while x < 5:
    print(x)
    x += 1

Functions

Functions allow code reuse and organization.

Defining a Function


def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

Types of Functions:

  • With Parameters: Accepts input values.
  • With Return Values: Outputs a result.
  • Anonymous (Lambda): A compact one-line function.

Object-Oriented Programming (OOP)

Python supports OOP principles such as classes and objects.


class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, my name is {self.name}")

p1 = Person("Alice", 25)
p1.greet()

File Handling

Python allows reading and writing files.


# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, World!")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)
