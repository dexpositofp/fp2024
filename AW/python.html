Introduction to Python

Python is a versatile and popular programming language known for its simplicity and readability. It is widely used in various fields, including web development, data analysis, artificial intelligence, scientific computing, and automation.

Why Use Python?

  • Easy to Learn: Python’s syntax is straightforward and user-friendly, making it an ideal language for beginners.
  • Community Support: Python has a vast and active community, providing numerous libraries and frameworks.
  • Versatility: Python can be used for small scripts or complex web applications.
  • Portability: Python programs can run on different operating systems with minimal changes.
  • Integration: Python can integrate with other languages like C, C++, or Java, making it extremely flexible.

Python Basics

1. Hello, World!

print("Hello, World!")

Run this code to display the text "Hello, World!" on the screen.

2. Variables and Data Types

x = 10  # Integer
y = 3.14  # Float
name = "Alice"  # String
is_student = True  # Boolean

print(x, y, name, is_student)

This example demonstrates variable declarations and basic data types in Python.

3. Control Structures

# If-Else Statement
age = 20
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

# For Loop
for i in range(5):
    print(i)

# While Loop
count = 0
while count < 5:
    print(count)
    count += 1

Control structures allow you to control the flow of your program.
